# Stranieri WEB - Copilot

Userscript Tampermonkey per supportare la lavorazione delle pratiche Stranieri WEB.

## Installazione

1. Aprire il file raw GitHub:
   `https://raw.githubusercontent.com/Cloud2129/Stranieri-Web---Copilot/main/stranieri-web-copilot.user.js`
2. Tampermonkey proporra' l'installazione dello script.
3. Confermare installazione.

## Note

La copia di distribuzione non include `file:///*`, usato solo per test locali.
